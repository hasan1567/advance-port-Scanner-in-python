# 📡 Advanced GUI Port Scanner (TCP, UDP, Nmap)

A professional Python GUI-based port scanner with support for TCP, UDP, and Nmap scans. Includes real-time banner grabbing, custom port presets, threading for fast scans, and an export feature to CSV or JSON.

---

## 🚀 Features

- ✅ TCP, UDP, and Nmap scan modes
- ✅ Aggressive vs Stealth scan modes
- ✅ Banner grabbing (optional)
- ✅ Custom port ranges & presets:
  - Top 100 Ports
  - Web Ports (80, 443, 8080)
- ✅ Export results as `.csv` or `.json`
- ✅ Dark mode UI
- ✅ Threaded scanning with progress bar
- ✅ Real-time logs with emojis 💻🔍🕵️

---

## 🛠️ Tech Stack

- Python 3
- Tkinter GUI
- Nmap (`python-nmap`)
- Socket Programming
- ThreadPoolExecutor (for multi-threading)
